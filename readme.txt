1. Создаем главный объект класса PiP
	- в конструкторе он создает 2 потока - 2 объекта класса VideoThread	
	- в конструкторе он создает виджет для отображени конечного видео
	- получает от двух потоков разные картинки в разное (или нет) время: Mat (?)	
	- соединяет их и показывает в виджете
2. Класс VideoThread 
	- инициализирует новый поток для обработки видео
	- в методе run(?):
		* создается VideoCapture, Mat, QTimer
		* считывается видео, его fps
		* по срабатыванию таймера новый кадр отправляется главному объекту
		

	

QThread должен быть использован как нормальный экземпляр потока: 
- подготовьте объект (QObject) класса со всей его функциональностью внутри.
- Потом создайте новый объект QThread
- положите в него QObject используя метод moveToThread(QThread*)
- вызовите start() от объекта QThread
